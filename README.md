# 📚 AI Study Topic Explainer

An AI-powered educational web application that helps students quickly understand complex study topics through simple explanations generated by Google's Gemini AI.

Instead of searching across multiple websites, students can enter a topic and instantly receive a clear, student-friendly explanation.

# 🧠 Project Overview

Students often struggle when they encounter a concept they do not understand. This project solves that problem by integrating an AI assistant that explains topics in simple language suitable for students.

The application allows users to:

• Enter a study topic  
• Send the topic to an AI model  
• Receive a simplified explanation instantly  

The goal of this project is to demonstrate **AI integration within modern web applications** using a full-stack JavaScript framework.

# ⚙️ Core Features

### 1️⃣ Topic Input
Students can enter any study topic such as:

• Photosynthesis  
• Newton's Laws  
• Binary Search  
• World War II  

---

### 2️⃣ AI Explanation Generation

The application sends the topic to the **Gemini AI model** and requests a simplified explanation.

Example prompt used:

Explain the topic "Photosynthesis" in simple terms for a student.


---

### 3️⃣ Explanation Display

The generated explanation is displayed clearly in a dedicated output section.

Example:

**Topic:** Photosynthesis  

**Explanation:**  
Photosynthesis is the process by which plants make their own food using sunlight, carbon dioxide, and water. The plant uses energy from sunlight to convert these into glucose, which helps it grow and survive.

---

### 4️⃣ Loading Indicator

While the AI response is being generated, the application displays:
Generating explanation...


This improves the user experience and provides feedback that the request is being processed.

---

### 5️⃣ Error Handling

If the API fails or the topic input is empty, the application provides clear user-friendly messages.

Example:
Please enter a topic to continue.
or
Failed to fetch explanation. Please try again later.

# 🔌 API Integration

The project uses **Google Gemini API (gemini-2.0-flash)** to generate explanations.

Example request flow:

1. User enters topic  
2. Frontend sends request to `/api/explain`  
3. Backend calls Gemini API  
4. AI generates explanation  
5. Response is returned to frontend  

Example API request:
POST/api/explain

Input:
{
"topic": "photosynthesis"
}

Output:
{
"explanation": "Photosynthesis is the process by which plants..."
}
